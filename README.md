# @react-native-ui-components/icons

Icons for React Native

## Installation

```sh
npm install @react-native-ui-components/icons
```

## Usage

```js
import {Icon, icons} from "@react-native-ui-components/icons";

// ...

<Icon name={icons.edit} />;
```
